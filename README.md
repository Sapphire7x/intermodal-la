# Intermodal LA

A map and route planner for car-free travel in Los Angeles.

Intermodal LA combines:

- 🚶 Walking
- 🚲 Metro Bike Share
- 🚇 Metro Rail
- 🚌 Local buses
- 🚉 Metrolink

to help users discover multimodal trips across the Los Angeles region.

## Goals

- Show bike + rail + bus routes
- Visualize Metro Bike Share stations
- Incorporate real-time bike availability
- Support multiple transit agencies
- Help people navigate Los Angeles without a car

## Built With

- Next.js
- TypeScript
- MapLibre
- OpenStreetMap
- GTFS
- GBFS
